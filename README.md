## Что делает

ElectroKit автоматически рассчитывает электрику квартиры, дома или офиса по нормам ПУЭ:

- **Разбивает нагрузки на группы** — освещение, розетки, выделенные линии, кондиционеры
- **Подбирает сечение кабеля** — ВВГнг с учётом типа линии и минимальных сечений по ПУЭ
- **Выбирает номинал автомата** — с коэффициентом запаса 1.25 для мощных нагрузок
- **Определяет мокрые зоны** — автоматически по названию комнаты, ставит дифавтоматы и УЗО
- **Считает падение напряжения** — ΔU% по длине линии, сразу при расчёте
- **Проверяет селективность** — ошибка если групповой автомат ≥ вводному
- **Генерирует схему щитка** — визуальная DIN-рейка с легендой
- **Составляет полную смету** — материалы и работы с ценами, итого в вашей валюте

## Флоу работы

1. Добавить комнаты и приборы
2. Указать расстояние от каждой комнаты до щитка
3. Нажать «Рассчитать»
4. Получить: группы с кабелями и автоматами, схему щитка, полную смету

## Что рассчитывает

**Группы нагрузки** — для каждой комнаты:
- Освещение → 1.5 мм², автомат по нагрузке
- Розетки и приборы → минимум 2.5 мм² + C16
- Стиральная машина, посудомойка, духовка, бойлер → минимум 2.5 мм², ток ×1.25
- Кондиционер → отдельная группа на каждый блок, минимум 2.5 мм², ток ×1.25

**Коэффициент спроса (Kd):**
- Квартира — 0.7
- Частный дом — 0.8
- Офис — 0.6

**Мокрые зоны** — по ключевым словам в названии:
`ван`, `санузел`, `туалет`, `кухн`, `баня`, `душ`, `kitchen`, `bathroom`, `wc`

**Падение напряжения:**
```
ΔU% = (2 × L × I × ρ) / (S × U) × 100
ρ = 0.0175 Ω·мм²/м (медь)
```

**Проверки:**
- Расчётный ток > номинал вводного → ⛔ перегрузка
- Групповой автомат ≥ вводному → ⛔ нарушение селективности
- Разница < 1 ступени → ⚠ низкая селективность
- Нагрузка > 80% вводного → 💡 рекомендация увеличить
- Розеточная группа > 3500 Вт или > 6 розеток → 💡 разделить
